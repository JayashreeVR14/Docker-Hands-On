# Flask and  MySQL Docker Networking Project

##  Overview
This project demonstrates Docker networking using:
- Flask (backend)
- MySQL (database)

##  Architecture
Flask container connects to MySQL container using Docker network.

##  Project Structure
- app/ → Flask application
- requirements/ → Python dependencies
- docker/ → Dockerfile
- procedure/ → Steps to run project
- output/ → Output and logs

##  How it Works
1. Create Docker network
2. Run MySQL container
3. Build Flask image
4. Run Flask container
5. Access via browser

##  Key Concepts
- Docker networking
- Container communication using container name
- Environment variables
- Backend + Database integration
